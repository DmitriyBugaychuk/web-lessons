Розмітка заголовків

Заголовки впорядковують текст, формуючи його візуальну структуру. При використанні заголовків необхідно враховувати їх ієрархію. В заголовки не можна вкладати інші елементи.
<h1> - заголовок найвищого рівня, на сторінці рекомендовано використовувати не більше одного разу.
<h2>, <h3>, <h4>, <h5>, <h6> - заголовки нижчих рівнів, розташовані за спаданням їх важливості.

Синтаксис:

          <h1> - Заголовок 1-го рівня </h1>  
          <h2> - Заголовок 2-го рівня </h2> 
          <h3> - Заголовок 3-го рівня </h3> 
          <h4> - Заголовок 4-го рівня </h4> 
          <h5> - Заголовок 5-го рівня </h5> 
          <h6> - Заголовок 6-го рівня </h6>

Результат:

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня

Теги для форматування тескту

<strong> - вказує на важливість фрагменту тексту, робить його напівжирним

Синтаксис:

<strong> Тьєррі Анрі </strong> - найкращий бомбардир збірної франції в історії футболу

Результат:

Тьєррі Анрі - найкращий бомбардир збірної франції в історії футболу

<em> - виділяє текст курсивом

Синтаксис:

<em> Тьєррі Анрі </em> - зіграв за футбольний клуб Арсенал 377 матчів

Результат:

Тьєррі Анрі - зіграв за футбольний клуб Арсенал 377 матчів

<small> - робить виділений текст меншого розміру на 1 одиницю

Синтаксис:

Тег "small" <small> робить виділений текст меншого розміру на 1 одиницю </small>

Результат:

Тег "small" робить виділений текст меншого розміру на 1 одиницю

<ins> - підкреслює текст

Синтаксис:

В професійній футбольній команді має бути мінімум <ins> 23 гравця </ins>

Результат:

В професійній футбольній команді має бути мінімум 23 гравця

<del> - закреслює текст

Синтаксис:

Знижка на товар тепер замість <del> 5999 </del> 5699 грн

Результат:

Знижка на товар тепер замість 5999 5699 грн

<sub> - вказує нижній індекс у виділеному місці

Синтаксис:

CH<sub>4</sub> - формула метану

Результат:

CH4 - формула метану

<sup> - вказує верхній індекс у виділеному місці

Синтаксис:

домашнє завдання розв'язати рівняння x<sup>2</sup> + 5x<sup>3</sup> = 10

Результат:

домашнє завдання розв'язати рівняння x2 + 5x3 = 10

Цитати, означення, аббревіатури

<code> - призначений для відображення одного або декількох рядків тексту, який є програмним кодом. Сюди відносяться імена змінних, ключові слова, тексти, функції тощо.

Синтаксис:

Тег "code" <code>призначений для відображення одного або декількох рядків тексту, який є програмним кодом. </code>

Результат:

Тег "code" призначений для відображення одного або декількох рядків тексту, який є програмним кодом.

<kbd> - використовується для позначення тексту, який набирається на клавіатурі або назви клавіш

Синтаксис:

Тег "kbd" <kbd> використовується для позначення тексту, який набирається на клавіатурі або назви клавіш </kbd>

Результат:

Тег "kbd" використовується для позначення тексту, який набирається на клавіатурі або назви клавіш

Синтаксис:

Тег "samp" використовується для відображення тексту, який є результатом <samp> виведення комп'ютерної програми чи скрипту </samp>

Результат:

Тег "samp" використовується для відображення тексту, який є результатом виведення комп'ютерної програми чи скрипту

<pre> - визначає блок попередньо форматованого тексту

Синтаксис:

Тег "pre" <pre> визначає блок попередньо форматованого тексту </pre>

 визначає блок попередньо форматованого тексту

<abbr> - вказує, що послідовність символів є абревіатурою

Синтаксис:

Тег "cite" позначає текст як цитату або виноску <cite>як цитату або виноску</cite> на інший матеріал

Результат:

Списки

<ul> - використовується для створення маркованого (невпорядкованого) списку <li> - визначає окремий елемент списку

Синтаксис:

<ul> <li> Англійська мова </li> <li> Комп'ютерна графіка </li> <li> Дискретна математика </li> </ul>

Результат:

  • Англійська мова
  • Комп'ютерна графіка
  • Дискретна математика

<ol> - визначає впорядкований (пронумерований) список

Синтаксис:

<ol> <li> Англійська мова </li> <li> Комп'ютерна графіка </li> <li> Дискретна математика </li> </ol>

Результат:

  1. Англійська мова
  2. Комп'ютерна графіка
  3. Дискретна математика
